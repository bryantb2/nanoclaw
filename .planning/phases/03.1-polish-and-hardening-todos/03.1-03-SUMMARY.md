## Pending: Tasks 3-4 (OAuth Token)

### Task 3 (checkpoint): Create Linear OAuth App
User must visit https://linear.app/settings/api/applications/new and create an OAuth application named "Fleet Bot". The token type needed is an **Actor Token** (not a user-level token) — this makes activity appear as the app.

Steps:
1. Go to https://linear.app/settings/api/applications/new
2. Create app named "Fleet Bot" (callback URL can be `http://localhost:3000/callback`)
3. After creating, go to the app's settings page
4. Under "Actor tokens" (or "API access"), generate a new actor token
5. Copy the token (starts with `lin_oauth_...` or similar)
